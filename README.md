##Wordly Dictionary SPA
Overview

Wordly Dictionary SPA is a modern Single Page Application (SPA) built using HTML, CSS, and JavaScript. The application allows users to search for English words and instantly retrieve their meanings from the Free Dictionary API without reloading the page.

The project demonstrates the use of JavaScript event listeners, DOM manipulation, Fetch API, dynamic styling, and interactive user features while providing a clean and responsive user experience.

###Features
- Search for English words
- Display word definitions
- View pronunciation and phonetic spelling
- Listen to the pronunciation using audio playback
- Display part of speech
- Show example usage (when available)
- Display synonyms (when available)
- Save favorite words
- Prevent duplicate saved words
-Dynamically highlight saved words using JavaScript
-Display an error message when a word cannot be found
-Single Page Application (SPA) experience with no page refresh

How It Works
Enter an English word into the search bar.
Click the Search button.
The application sends a request to the Free Dictionary API.
The returned data is displayed on the page, including:
Word
Pronunciation
Part of Speech
Definition
Example
Synonyms
Click Save Word to add the word to your saved words list.
Click Pronounce to hear the correct pronunciation of the word.
