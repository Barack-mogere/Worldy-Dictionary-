## Wordly Dictionary SPA

A Single Page Application (SPA) that allows users to search for English words and retrieve dictionary information in real time using the Free Dictionary API. The application updates content dynamically without reloading the page, providing a fast and interactive user experience.


### Features
Search for any English word
View word definitions
Display phonetic pronunciation
Listen to audio pronunciation (when available)
View the part of speech
Display example usage
Show synonyms (when available)
Save favorite words
Prevent duplicate saved words
Display user-friendly error messages for invalid searches
Dynamically update the interface using JavaScript

### Technologies Used

HTML5
CSS3
JavaScript (ES6)
Fetch API
Free Dictionary API

### API

This application uses the Free Dictionary API to retrieve dictionary data.

API Endpoint:

https://api.dictionaryapi.dev/api/v2/entries/en/<word>

Example:

https://api.dictionaryapi.dev/api/v2/entries/en/hello
